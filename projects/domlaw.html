Why I Made DomLaw

DomLaw is based on a company that does business in Nigeria, It is a multitasking product and services company. I undertook this project after spending some months away from doing anything web related. I felt I needed a project that I'll use to get back to speed with the Javascript, React and the general web development space. While making this app, I was more aware of how the application's memory footprint can help speed up or slow down the entire app. I was more concern for the general efficiency of my programs.

Choosing NextJS for this project was based on its native support for page routing, which meant that I did not have to use other dependencies for that, NextJS also offered static rendering of React applications and other optimizations which improves the app speed and resource loading like images and styles.

Getting the application to look the same on different browser was a challenge, also finding a balance between appearance and accessibility was one of the issues that I had to content with. Now about the name, DomLaw is Lawrence Dominic.
